# Ryvus Flow

Orchestration & persistence layer that sits above `ryvus-engine`.

## Features (initial)

- Pipeline registry and manager
- Flow context with pluggable state store
- In-memory state store implementation
- Extension points for triggers

## Quick start

```bash
cargo run --example hello
```
